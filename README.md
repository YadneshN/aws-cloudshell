# Master the Cloud CLI with AWS CloudShell

This repository contains **step-by-step AWS CLI labs** designed to be run **entirely from AWS CloudShell**.

## Services Covered
- CloudShell Environment & AWS CLI
- IAM (Users, Policies, Roles)
- Amazon S3 (Bucket lifecycle)
- Amazon EC2 (Free Tier – create, start, stop, terminate)

## Prerequisites
- AWS Account
- IAM permissions to use IAM, S3, EC2
- AWS CloudShell access

## How to Use
1. Open AWS Console
2. Launch CloudShell
3. Follow labs in order

> ⚠️ Always clean up resources to avoid charges.
